Nscc Database Information

The use of the code first approach to building database models using the entityframework. The resulting database is then populated and a webservice is built for the database using Odata.The service is then pushed to azure as a rest api.

  • Build the NSCC Application Form Database using Entity Framework
  • Generate the SQL script and POCO objects resulting from the creation of the Model.
  • Add the provided starting data to your NSCC Application Database
  • Create an OData Web Service for your NSCC Application Form model and database which will expose CRUD functionality over HTTP using ASP.net WEB API.
  • Publish the Web Service and populated database to Azure.
