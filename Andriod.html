Quiz App Info

Android application in portrait mode that allows a user to prepare for ‘matching-style’ quizzes. When the program is running, it will display one term and four possible definitions; one of which is the correct one. The user will then select which definition they think matches the term displayed. The program will indicate if the user has made a correct or incorrect selection by displaying a toast message (or other means e.g. colors, etc.) to that affect.

  • Include exactly ten term/definition pairs for testing and demonstration purposes.
  • Provided with the same quiz data, the program should display the definitions in a different order from run to run.
  • The program should use at least one ArrayList to hold collections of items and at least one HashMap to hold name-value pairs.
